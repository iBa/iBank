iBank
=============

Current Features
-------
* Regions  (custom percentages per region, bound to region, ...)
* BankAccounts (custom interests + balance)
* Loan money
* Supports UUIDs (fully supports name changing)
* All fully configurable
  
Supported Database Types
-------
* Mysql
* SQLite
